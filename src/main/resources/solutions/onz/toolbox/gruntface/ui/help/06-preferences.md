# Preferences

## Theme

**View → Theme** lets you choose:

- **Light** — always light.
- **Dark** — always dark.
- **Auto** — follow the operating system's color scheme. If the OS scheme
  changes, GruntFace switches with it.

## Location overrides

GruntFace auto-detects your Terraform modules folder and your commons folder
from the project layout. If auto-detect picks the wrong folder, use
**View → Set modules location…** or **View → Set commons location…** to point
at the right one. The overrides persist between sessions until you clear them
with **View → Clear location overrides**.

## What GruntFace remembers

Between sessions, GruntFace remembers:

- The last Terragrunt project you opened.
- Your modules and commons location overrides.
- The main window size.
- The User Guide window size and position.
- Whether **Show modules** and **Show includes** were on.
- Your theme choice.
